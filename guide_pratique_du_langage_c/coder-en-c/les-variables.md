# Les variables

Les variables sont des conteneurs utilisés pour stocker des valeurs qui seront utilisées pendant l'exécution du programme. Le nom des variables est arbitraire, tant qu'il suit certaines règles de nommage.

## **Déclaration de variables**

Les systèmes informatiques nécessitent un espace de stockage temporaire pour les données lors de l'exécution de calculs ou de traitements. Dans la programmation en C, cela correspond à la déclaration d'une variable qui réserve un emplacement dans la mémoire de l'ordinateur.

Une variable en C comporte deux éléments essentiels :

* Un type de données : Il définit la nature des données que la variable peut contenir. Par exemple, cela pourrait être un caractère (`char`), un nombre entier (`int`), un nombre réel (aussi appelé nombre à virgule flottante, abrégé en `float`), etc. Pour spécifier le type d'une variable, nous utilisons un mot-clé approprié correspondant à chaque type de données.
* Un identificateur : C'est le nom de la variable.
